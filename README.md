# College Mapping System

A web-based campus navigation project developed for NMAM Institute of Technology (NMAMIT) to help students locate academic blocks, departments, laboratories, and classrooms.

## Features

* Information about major academic blocks
* Floor-wise lab and department details
* Dedicated page for each block
* Sign In and Sign Up interface
* Responsive user interface

## Built With

* HTML
* CSS
* JavaScript
