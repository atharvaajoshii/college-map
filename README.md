College Mapping System

A web-based campus navigation project designed to help students explore academic blocks, departments, laboratories, and facilities within NMAM Institute of Technology.

Live Demo

Website:
[College Mapping System](https://atharvaajoshii.github.io/college-map/?utm_source=chatgpt.com)

Overview

Finding classrooms, laboratories, and departments in a large campus can be challenging, especially for first-year students and visitors. This project provides a simple and interactive way to explore different academic blocks and view information about the facilities available on each floor.

The system includes details about major blocks in NMAMIT such as:

C. V. Raman Block
APJ Abdul Kalam Block
Sir M. Visvesvaraya Block
V. Rajaraman Block
S. Ramanujan Block
Atal Block

Each block contains floor-wise information about laboratories, classrooms, and departments.

How It Works
Users visit the landing page.
Navigation options allow users to sign in or create an account.
After authentication, users can access the campus map.
Selecting a block displays detailed information about:
Departments
Laboratories
Classrooms
Floor layouts
Users can quickly locate facilities without manually searching the campus.
Motivation

Large academic campuses can be difficult to navigate for new students. This project was developed to provide a centralized platform where students can quickly find information about buildings and facilities, reducing confusion and improving accessibility.
